# FreeCodeCamp React with Authentication without Server

This is a project that use:

* [Vite](https://vitejs.dev/)
* [Wayne](https://github.com/jcubic/wayne)
* [React](https://react.dev/)

## Installation

```
npm install
```

## Build

To build and run the project use:

```bash
npm run build
npm run serve
```
